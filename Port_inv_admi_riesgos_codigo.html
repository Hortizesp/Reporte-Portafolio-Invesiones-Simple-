Portafolio de inversion¶

Para este portafolio de inversiones se usaron las acciones de: 'ICHB.MX', 'CEMEXCPO.MX', 'BACHOCOB.MX' y 'GFINBURO.MX' . Se midio el rendimiento de las acciones, se obtuvo sus matrizes de Marginalidad , Covarianza, Correlacion y Markowitz. Ademas de clacular el mejor portafolio siguiendo la minima varinza que establece buscar maximizar el rendimiento esperado de las inversiones mientras minimizan el riesgo. En este marco, la varianza del rendimiento de un portafolio se utiliza comúnmente como medida de riesgo. Por lo tanto, un portafolio de mínima varianza es aquel que ofrece el menor riesgo posible dado un nivel de rendimiento esperado.

def get_adj_closes(tickers, start_date,end_date ):
    adj_closes = pd.DataFrame()
    for ticker in tickers:
        stock = yf.Ticker(ticker)

        hist = stock.history(start=start_date, end=end_date)
        adj_closes[ticker] = hist['Close']
    return adj_closes
